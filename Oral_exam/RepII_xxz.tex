The course name is ``Representation II", and the examiner is Prof. Stroppel. 

Like anyone else, I was asked to begin with one topic. I begin with the Coxeter group.
\begin{itemize}
\item definition;
\item define $(W_P,S_P)$. Is that obvious that $(W_P,S_P)$ is a Coxeter group? (No)
\item show that by geometrical representation. (define geometrical representation) Key ingredient is to distinct two different $W_P$.
\item give an example of Coxeter group which is not finite. Can you see that group?
\end{itemize}

Then we work on Hecke algebras.
\begin{itemize}
\item define generic Hecke algebra $\mathcal{H}_v(W)$ for a Coxeter group;
\item find a basis, show the well-defineness, and why it is a basis;
\item for the linear independent part, define Hecke algebra $H(\GL_n(\mathbb{F}_q),B)$;
\item state the Bruhat decomposition and sketch a proof;
\item state the KL-basis;
\item compute $T_s * T_s$; (explains the relation in $\mathcal{H}_v(W)$)
\end{itemize}

We switch to the finite irreducible representation of $\GL_n(\mathbb{C})$.

\begin{itemize}
\item Do we have a classification for that? (rep <-> dominant weight)
\item Why is that surjective? (construction, some cases to check)
\item How to find a highest weight? (reduced to finding a weight, reduced to finding a common eigenvectors, which uses the Borel fixed point theorem)
\item State the Borel fixed point theorem, and give a sketsch of proof. (Use one fact concerning about solvable group and complete varieties)
\end{itemize}


Since I mentioned about $G/G_x$ in the last question, we go to part of varieties.
\begin{itemize}
\item For an affine algebraic group, is any closed subgroup also affine?
\item Show that affine algebraic group is linear.
\item The same method can be applied to what problem? (The existence of Chevalley quotient)
\item Does proper apply projective?
\item Is there any example of affine and proper variety? (points) Other examples? (I believe no)
\item By using the idea of projective apply proper, show that $\mathbb{P}^1$ is not affine. (I didn't understand that question, and said that we just show that by computing the global sections. The final answer is “to shoot a mosquito with an anti-aircraft gun”, and she expected me to say the fact that affine and proper variety can be only finite many points.)
\end{itemize}

Finally we come back to Borel-Weil theorem.
\begin{itemize}
\item State that theorem and give a sketch of proof quickly. (I give the geometrical proof, which uses the Hartdog's lemma.)
\end{itemize}
